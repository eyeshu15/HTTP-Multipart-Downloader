					HTTP DOWNLOAD USING RANGE REQUESTS

Language used : JAVA

The parts_i files are the files downloaded using Range request.

The "Combined_file" is the files obtained from the parts_i.

The "originalFile_for_sha" is the actual image downloaded from the url to perform sha256 with the Combined_file.

To COMPILE the code enter this in ur cmd or terminal:-

		java -jar http_downloader.jar <url> <number_of_connections>
